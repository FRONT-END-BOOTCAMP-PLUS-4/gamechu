## ✨ 프로젝트 소개

> **겜추(GameChu)**는 게이머를 위한 **게임 추천 서비스**와 **커뮤니티 기능**을 통합한 웹 플랫폼입니다.

- **로튼토마토식 평점 시스템**을 도입해 게임에 대한 객관적인 평가 제공
- **유저 간 1:1 논쟁(투기장)** 기능으로 게임에 대한 다양한 관점을 공유
- **티어 및 활동 점수 시스템**을 통해 활발한 유저 참여 유도

## 🧩 주요 기능

### 📌 게임 추천

<p align="center">
<img src="https://github.com/user-attachments/assets/23cb9ac4-0c12-4524-b846-b310f907beef" width="700"/>
</p>

- 다양한 플랫폼, 장르, 테마별 게임 탐색 가능
- 게임 상세 페이지 제공 (설명, 출시일, 스크린샷 등)
- **겜잘알 리뷰** vs **일반 리뷰**로 리뷰 분리
- 리뷰 좋아요 기반으로 신뢰도 판단

### 🧠 투기장 (Arena)

<p align="center">
  <img src="https://github.com/user-attachments/assets/ad978e12-3526-41ba-b2cd-59f6674d76f8" width="700" />
</p>

- 유저 간 1:1 실시간 논쟁
- 제한된 채팅 횟수 및 글자 수
- 종료 후 투표 기능 및 결과 공개

### ⭐ 유저 티어 및 활동 점수 & 마이페이지

<p align="center">

- 누적 점수로 유저 티어 설정 → 활동성 시각화
- 위시리스트 관리  
- 내가 쓴 리뷰, 참여한 투기장 기록  
- 나의 티어, 포인트 히스토리 확인
